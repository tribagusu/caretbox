# Caretbox

A centralized developer knowledge hub for code snippets, AI prompts, docs, commands, and more.

## What it is

Developers keep their essentials scattered across VS Code, Notion, bash history, bookmarks, and random folders. Caretbox gives you one searchable, AI-enhanced hub for all your dev knowledge and resources.

**Supported item types:** Snippet · Prompt · Note · Command · File · Image · URL

## Tech Stack

- **Framework:** Next.js (React 19) with TypeScript
- **Database:** Neon PostgreSQL + Prisma ORM
- **Auth:** NextAuth v5 (email/password + GitHub OAuth)
- **Storage:** Cloudflare R2 (file/image uploads)
- **UI:** Tailwind CSS v4 + shadcn/ui
- **AI:** OpenAI (auto-tagging, summaries, explain code)
- **Deployment:** PaaS
